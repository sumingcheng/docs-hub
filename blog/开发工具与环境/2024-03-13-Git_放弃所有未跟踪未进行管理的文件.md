# Git 放弃所有未跟踪（未进行管理）的文件



 



公司前端项目进行文件结构上的大调整，导致有巨多 `node_modules` 里面的文件没有被跟踪，所以就要额外的去清理，否则连文件都无法提交，除非进入到你要提交文件的目录下，才可以正常的提交。所以就要删除这些未跟踪的文件。

### 查看哪些文件要被删除  
```
git clean -n
```
### 删除未跟踪的文件  
```
git clean -f
```
### 如果还包括未跟踪的目录，请使用  
```
git clean -fd
```
