# Git命令行提交特定文件夹下的内容



 **Link:** [https://zhuanlan.zhihu.com/p/686865578]


```
git add <folder_path>/*  # 添加特定文件夹下的所有更改
git commit -m "Your commit message"  # 提交这些更改
git push  # 将提交推送到远程仓库
```
