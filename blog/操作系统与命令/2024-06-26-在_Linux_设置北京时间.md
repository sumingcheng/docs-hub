# 在 Linux 设置北京时间



 **Link:** [https://zhuanlan.zhihu.com/p/705563097]

## 查看时区  
```
timedatectl
```
## 查看所有可用时区  

会输出很多

```
timedatectl list-timezones
```
## 设置北京时间  
```
sudo timedatectl set-timezone Asia/Shanghai
```
